# POOSD-Group-Project
Group 12's project for Processes for Object Oriented Software Development.

## What is it?
This application is a contact manager with basic capabilities.

## Features
- Login and registration so that contacts are linked to the user they belong to
- The following contact information will be saved: full name, email, phone number, and the date the contact was created on
- Users will be able to create, read, search through, update, and delete contacts

## Project specifications
- This web app resides on a remote server (DigitalOcean)
- We are using a LAMP stack
- The front end communicated with the back end through the use of APIs
- The application data is stored in an MySQL database
